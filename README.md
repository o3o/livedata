# livedata
Proof of concept di visualizzazione dati in csv


## Installazione dipendenze del progetto
Il modo più semplice è:
```
$ npm install
```

Per elencare i pacchetti globali installati
```
npm list -g --depth=0
npm list  --depth=0  // locali
```

## Come compilare
- `dub build` oppure `dubbio`
